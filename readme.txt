Description

This project focuses on performing ETL (Extract, Transform, Load) operations on two CSV files downloaded from Kaggle. It utilizes libraries such as Pandas, Matplotlib, SQLite3, NumPy, and Seaborn. The reference links can be found at the end of this document.


Dataset Analysis

The project analyzes datasets on income from music album sales by the world's top musicians. It explores the genres, technical data of albums, and provides in-depth insights into well-known artists. Detailed graphs and a SQL database are generated for comprehensive analysis.

Descripción

Este proyecto se enfoca en realizar operaciones ETL (Extract, Transform, Load) en dos archivos CSV descargados desde Kaggle. Se utilizan librerías como Pandas, Matplotlib, SQLite3, NumPy y Seaborn. Los enlaces de referencia se encuentran al final de este documento.

Análisis del Conjunto de Datos

El proyecto analiza conjuntos de datos sobre ingresos por venta de discos de los mejores artistas musicales del mundo. Explora los géneros, datos técnicos de los discos y proporciona información detallada sobre artistas reconocidos. Se generan gráficas detalladas y una base de datos SQL para un análisis completo.
